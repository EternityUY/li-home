# Media 文件夹

此文件夹用于存放音频文件。

## 需要的文件

### 1. music.mp3
- **位置**: `./media/music.mp3`
- **说明**: 背景音乐主文件（MP3 格式）
- **格式**: MP3
- **建议**: 
  - 使用压缩率适中的 MP3（128-192 kbps）
  - 建议使用纯音乐或轻音乐，避免突兀的节拍
  - 文件大小建议控制在 3-5MB 以内以优化加载速度

### 2. music.ogg（可选）
- **位置**: `./media/music.ogg`
- **说明**: 背景音乐备选格式（OGG 格式，用于更好的浏览器兼容性）
- **格式**: OGG
- **建议**: 
  - 与 MP3 文件内容相同
  - 提供此格式可以确保在 Firefox 等浏览器中获得最佳兼容性

## 使用说明

1. 将你的音频文件放入此文件夹
2. 确保文件名与代码中的引用一致：
   - `music.mp3` 
   - `music.ogg`（可选）

3. 代码会自动加载并使用这些文件作为背景音乐

## 注意事项

- 请确保你使用的音乐文件符合版权要求
- 建议使用较短的循环音乐片段（30秒-2分钟）
- 如果音频文件加载失败，播放器会显示错误信息
- 文件不存在时，播放器功能仍然可用，只是无法播放音频

## 音频格式转换

如果需要将音频转换为不同格式：
- **转 MP3**: 使用 Audacity、FFmpeg 等工具
- **转 OGG**: 使用 VLC、FFmpeg 等工具

### FFmpeg 转换示例：
```bash
# 转换为 MP3
ffmpeg -i input.wav -codec:a libmp3lame -b:a 192k music.mp3

# 转换为 OGG
ffmpeg -i input.wav -codec:a libvorbis music.ogg
```
