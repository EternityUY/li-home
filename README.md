## 🧭 关于我

你好，我是 li，一位热爱感性与理性交织的探索者。这个主页是我在数字世界中的一片小天地，用来记录我喜欢的事物、思考的片段，以及那些让我心动的瞬间。

我喜欢：

- 🎐 **二次元文化**：动漫、轻小说、角色设定与世界观构建
- 🎧 **轻音乐**：钢琴、吉他、环境音，适合夜晚沉思或清晨醒来
- 🎮 **Galgame**：剧情驱动的互动体验，感受角色的情感与选择的重量
- 🌄 **风景摄影与欣赏**：城市、自然、虚构世界的风景都让我着迷
- 🧠 **深度问题**：哲学、心理、技术、社会议题，喜欢思考无解的问题
